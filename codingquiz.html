Coding Quiz Game, 2025

Role: UX/UI Design, Front-End Development, Interaction Design
Deliverables: Responsive Web App, Question Logic, Leaderboard System, UI Components

View Live Project

The Coding Quiz Game was designed as an interactive way to help new developers test and strengthen their JavaScript knowledge. The goal was to make learning engaging through timed challenges, real-time feedback, and score tracking.

Problem

Many online coding quizzes feel flat and uninspiring — they lack interactivity, time pressure, or meaningful feedback that helps learners stay motivated. Users needed a fast, fun, and rewarding way to gauge their coding knowledge.

Solution

The solution was a clean, dynamic quiz app that merges play and practice. I designed and developed a fully responsive web quiz where users answer code-related questions under a countdown timer, earning instant feedback and final scores.

Features

  • Interactive Gameplay: Real-time question and timer updates for each round.
  • Progress Tracking: Dynamic progress bar to visualize quiz completion.
  • Highscore System: Persistent leaderboard using local storage for user initials and scores.
  • Clean UI: Minimal design with focus on readability and responsiveness.
  • Exit Option: Users can exit mid-quiz, creating flexibility in user flow.

Impact

The Coding Quiz encourages users to stay engaged while testing their JavaScript fundamentals. By combining gamification with learning, it motivates users to practice more frequently and track their progress over time.
